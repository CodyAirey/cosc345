std::vector< std::string > get_pos_tags ( const std::string &  filename)

Fetch POS (Part-Of-Speech) tags for a given text file.

This function runs an external command (a compiled python POS tagger binary) on the input file and returns the POS tags as a vector of strings. The external command is OS-dependent.

Parameters
filenameThe path to the input text file.
Returns
A vector containing the POS tags.
Note
This function assumes that the tagger outputs a file with the ".pos" extension and that each tag in this file is separated by whitespace.
